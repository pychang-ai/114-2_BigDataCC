=== 任務 1：探索 Linux 目錄結構 ===
執行以下指令：
ls /

（貼上結果）
bin                etc   lib.usr-is-merged  mnt   run                 srv  var
bin.usr-is-merged  home  lib64              opt   sbin                sys
boot               init  lost+found         proc  sbin.usr-is-merged  tmp
dev                lib   media              root  snap                usr
請從上面的結果中，說明以下 5 個目錄各是做什麼的：
/home：存放一般使用者的個人資料與設定檔。
/etc：存放系統所有的組態設定檔。
/var：存放系統執行過程中會「動態變化」的檔案。最常見的是系統日誌 (Logs)、緩存 (Cache) 以及資料庫檔案。
/tmp：供系統或程式存放臨時檔案的地方。通常在系統重新開機後，這個目錄下的內容會被自動清空。
/usr：存放系統安裝的應用程式、指令碼與函式庫。

=== 任務 3：chown 與 chgrp 實作 ===
請依序執行以下指令並貼上每步的結果：

步驟 1：建立測試檔案並查看預設擁有者
touch ~/week06_test.txt
echo "Ownership Test" > ~/week06_test.txt
ls -l ~/week06_test.txt
（貼上結果，注意看擁有者和群組欄位）
-rw-r--r-- 1 u306 u306 15 Mar 31 16:16 /home/u306/week06_test.txt
步驟 2：建立測試用的使用者和群組
sudo useradd -m tempuser
sudo groupadd tempgroup
（貼上結果）
無輸出
步驟 3：用 chown 改變擁有者
sudo chown tempuser ~/week06_test.txt
ls -l ~/week06_test.txt
（貼上結果，觀察擁有者欄位的變化）
-rw-r--r-- 1 tempuser u306 15 Mar 31 16:16 /home/u306/week06_test.txt
步驟 4：用 chgrp 改變群組
sudo chgrp tempgroup ~/week06_test.txt
ls -l ~/week06_test.txt
（貼上結果，觀察群組欄位的變化）
-rw-r--r-- 1 tempuser tempgroup 15 Mar 31 16:16 /home/u306/week06_test.txt
步驟 5：用 chown 一次改回自己
sudo chown $(whoami):$(whoami) ~/week06_test.txt
ls -l ~/week06_test.txt
（貼上結果，確認擁有者和群組都改回來了）
-rw-r--r-- 1 u306 u306 15 Mar 31 16:16 /home/u306/week06_test.txt
步驟 6：清理
rm ~/week06_test.txt
sudo userdel -r tempuser
sudo groupdel tempgroup
（貼上結果）
userdel: tempuser mail spool (/var/mail/tempuser) not found
